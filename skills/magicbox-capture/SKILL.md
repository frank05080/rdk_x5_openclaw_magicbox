# MagicBox Capture Skill
Make MagicBox capture image

## Commands
- `/userdata/MagicBox/.roo/skills/magicbox-capture/peripheral_skill.py /capture <path>`: Start capturing

### Parameters
- `path`: image saving path

### Example
`/userdata/MagicBox/.roo/skills/magicbox-capture/peripheral_skill.py /capture /userdata/MagicBox/images/cam.jpg` - Start capturing
